== Description ==

Wordpress captcha contact form with frontend TinyMCE editor is a widget which allows to display , on your wordpress website , a customizable contact form  .

Wordpress captcha contact form with frontend TinyMCE editor widget is distinguished by a prefix to avoid conflicts in database Options table .

= [ CUSTOMIZE the CAPTCHA CONTACT FORM EDITOR ] =

Wordpress captcha contact form with frontend TinyMCE editor uses javascripts of the famous GNU LGPL TinyMCE Editor .

**Regarding plugin administrator** , Wordpress captcha contact form with frontend TinyMCE editor gives him the following opportunities :

* he can add **up to 4 buttons lines** which allows user to format its text ( Buttons list is available at the following address : 
  [http://donicaptcha.donimedia-servicetique.net/?p=85](http://donicaptcha.donimedia-servicetique.net/?p=85 "List of TinyMCE editor buttons available with DoniCaptcha plugins") 
* he can choose editor **toolbar location** ( above or below captcha contact form text area )
* he can choose editor **toolbar alignment** ( Left, Center or Right )
* he can choose editor **statusbar location** ( None , above or below captcha contact form text area )
* he can authorize or not **Textarea resizing**
* he can choose between **3 types of editor skins**

**Regarding  user** , Wordpress captcha contact form with frontend TinyMCE editor gives him the  opportunity of enriching its message by :

* Adding **images**
* Embedding **media**
* Inserting **special characters**
* Pasting its message from a **Word document**
* And much more ..

Besides , Wordpress captcha contact form with frontend TinyMCE editor is user-friendliness since user can **toggle full screen mode** and write more easily .
Finally , Wordpress captcha contact form with frontend TinyMCE editor allows user to keep a copy of its message **by printing it** .

= [ WIDGET INSTALLATION PROCEDURE ] =


**1. WIDGET INSTALLATION :**

* Download Wordpress captcha contact form with frontend Tiny Mce editor widget to your computer , from official Wordpress directory .
* Launch your web browser and connect you to your Wordpress blog .
* Click on the plugins menu .
* Click on the Add New button .
* Click on the Upload link .
* Use the browse button to select the plugin zip file that was downloaded .
* Click on the Install Now button .
* Once the widget is installed, click on the Activate plugin button .
* The widget is now ready to use !
 

**2. WIDGET CONFIGURATION :**

* You can access Wordpress captcha contact form with frontend Tiny Mce editor widget from  Appearance --> Widget menu or from widgets option of your theme .
* Drag and drop Wordpress captcha contact form with frontend Tiny Mce editor widget towards the appropriate sidebar .
* Enter a title for this widget .
* Translate the predefined labels  ( captcha contact form labels and error messages )  , at your convenience .
* Enter your e-mail address in contact form e-mail reply field ( required parameter ) .
* Thanks to list of Tiny Mce buttons available with DoniCaptcha plugins ( [http://donicaptcha.donimedia-servicetique.net/?p=85](http://donicaptcha.donimedia-servicetique.net/?p=85 "List of TinyMCE editor buttons available with DoniCaptcha plugins") ) , fill in editor buttons lines ( up to 4 lines ) .

**ATTENTION :**

* Each editor button name , drawn from the list , must be seperated with commas .
* Each editor buttons line must be filled in consecutively ( no empty lines between two filled in lines )
* To delete a buttons line , leave it empty , taking into account previous remarks
* You can manage all other parameters , at your convenience (captcha contact form editor toolbar location , captcha contact form statusbar toolbar location , captcha contact form inputs font size , captcha contact form toolbar alignment , captcha contact form textarea resizing, captcha contact form editor skins )
* Once your web page is refreshed , you can see that Wordpress captcha contact form with frontend Tiny Mce editor widget is now ready to use !
